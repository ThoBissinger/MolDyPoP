\doxysubsection{Detailed Description}
Main-\/file. Every computation starts here. 

The main-\/\+File progresses as follows\+:


\begin{DoxyEnumerate}
\item Read the file at input/infile.\+in (called infile for short)
\item Initialize parameter class object par
\item Read the infile into par. In case of input errors, abort the run. Return value is the error code thrown by the read function.
\item Depending on the value of the input parameter mode, choose
\begin{DoxyItemize}
\item mode = none. Do nothing.
\item mode = test. User-\/specified tests can be performed.
\item mode = integrate, mode=integrate\+\_\+cont, mode=equilibrate. Run specific integration routines from the routines namespace.
\item mode = samp. Run specific sampling routines from the routines namespace.
\end{DoxyItemize}
\item End the run, return value is the return value of the routine performed.
\end{DoxyEnumerate}
